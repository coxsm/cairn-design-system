Coin icon plus the three GP / SP / CP tally rows used for a party's wealth.

```jsx
<CurrencyPanel gp={12} sp={4} cp={0} />
```

The original printout hand-letters "GP / SP / CP" directly into the coin illustration; here they're three independent rows so amounts can be edited without touching art. Leave a row blank (`""`) to render an empty rule, matching an unfilled paper sheet.
